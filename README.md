# rust-microbit-v2-template

This is a template for creating Rust applications for the micro:bit V2.

## Commands

embed the application to the micro:bit:

```bash
cargo embed
```

## crate

* cortex-m
* cortex-m-rt
* embedded-hal
* microbit-v2
* nrf52833-hal
* nrf52833-pac
* panic-halt
* rtt-target
